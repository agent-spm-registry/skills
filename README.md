# skills

asdafd

## Overview

This is a rules-only SPM skill. It contains AI instructions (SKILL.md) that get
copied directly into your AI agent's configuration directory.

No code is executed — this skill only provides guidance to AI agents.

## Install

```bash
spm pull skills
```

## Supported Agents

Works with all 55+ agents: Cursor, GitHub Copilot, Cline, Gemini CLI, and more.

## Customize

Edit `SKILL.md` to update the instructions, then publish:

```bash
spm push
```
